ЛР1 — Статична верстка

Тут використовуються Flexbox та Grid Layout для розміщення блоків сторінки. Цей текст спеціально зроблено довшим, щоб продемонструвати функціонал кнопки "читати далі". Flexbox допомагає легко вирівнювати елементи в одному напрямку, тоді як Grid Layout дозволяє створювати складні двовимірні сітки для більш комплексного розміщення контенту на сторінці. Обидва підходи є ключовими для сучасної веб-розробки та адаптивного дизайну.
